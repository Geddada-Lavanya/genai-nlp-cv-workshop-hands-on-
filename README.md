# GenAI-NLP-CV Workshop Hands-on

## 📌 Overview

This repository contains all the hands-on work completed during a **3-day workshop** on:

* Generative AI (GenAI)
* Natural Language Processing (NLP)
* Computer Vision (CV)

The workshop focused on practical implementation and understanding of modern AI technologies through mini-projects and examples.

---

## 📅 Workshop Summary

### 🔹 Day 1: Generative AI

* Introduction to Generative AI
* Basics of AI models
* Simple text generation tasks
* Understanding how AI generates content

### 🔹 Day 2: Natural Language Processing (NLP)

* Text preprocessing (lowercase, stopwords removal, punctuation removal)
* Tokenization and basic NLP techniques
* Named Entity Recognition (NER)
* Mini NLP projects

### 🔹 Day 3: Computer Vision (CV)

* Image processing basics
* Face detection using OpenCV
* Working with images and real-time detection
* Mini CV-based applications
